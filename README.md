## Snail Part-time Job Network System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/22a58d2c30a946259813642abcef7cf9/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Enterprise, Administrator.

**User Functions:**
1. Login
2. Register
3. Part-time Job Information
4. Message Feedback
5. Personal Center (Change Password, Personal Information)
6. Job Application

**Enterprise Functions:**
1. Login
2. Register
3. Personal Center (Change Password, Personal Information)
4. Part-time Job Information
5. Job Application Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Enterprise Management
5. Part-time Job Information Management
6. Job Application Management
7. Message Board Management
8. Carousel Management
